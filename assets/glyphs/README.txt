Jianzipu glyph images (SVG)
===========================

These per-technique jianzipu glyph drawings are from the neuralfirings/JianZiPu
project (https://github.com/neuralfirings/JianZiPu), MIT-licensed, self-hosted
here so the reference renders real glyph artwork inline with no external requests.

File -> technique mapping:
  lg_mo 抹 | lg_tiao 挑 | lg_gou 勾 | lg_ti 剔 | lg_da 打 | lg_zhai 摘
  lg_pi 擘/劈 | lg_tuo 托 | xl_cuo 撮 | lg_lun 輪 | lg_banlun 半輪
  xl_bo 撥 | xl_la 剌 | lg_qiaqi 掐起 | lg_yan 罨
  mod_chuo 綽 | mod_zhu 注 | mod_fanyin 泛
